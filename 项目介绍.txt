技术栈：Vue 3 + TypeScript + Tailwind CSS + Pinia + Vite | Python + FastAPI + SQLAlchemy + MySQL | LangChain + Chroma 向量数据库 | JWT 认证 + bcrypt 加密

夜记 — AI 智能日记管理系统

项目描述：一款支持多用户的全栈 Web 日记平台，核心亮点为基于 ReAct Agent 架构的 AI 日记分析系统。用户可记录日记、管理标签、配置个人 LLM 模型，系统通过 LangChain ReAct Agent 结合 RAG 语义检索对日记内容进行深度分析并生成温暖回应。

核心职责与实现：
- 设计并实现 ReAct Agent 架构的 AI 分析引擎，集成 LangChain 工具链（日记语义搜索、天气查询），根据日记长度和历史数量智能切换 Chain/Agent 模式，优化 Token 消耗
- 基于 Chroma 向量数据库 + text2vec-base-chinese Embedding 模型构建 RAG 检索系统，实现按用户隔离的 Collection 设计，支持语义级别的历史日记检索，替代传统 SQL LIKE 关键词匹配
- 实现多 LLM 提供商适配（DeepSeek、OpenAI、通义千问等），用户可自行配置模型，API Key 采用 Fernet 对称加密存储，任何接口不返回原始密钥
- 设计智能防重分析机制，通过日记内容长度比对避免重复调用 LLM，节省 Token 费用；记录详细的 Token 消耗明细（缓存命中/付费输入/付费输出）
- 使用 FastAPI 构建 RESTful API，实现完整的 JWT 认证体系（注册/登录/Token 刷新），数据严格按用户隔离；采用分层架构（Router → Service → Model）保证代码可维护性
- 前端采用 Vue 3 Composition API + TypeScript 开发，使用 Pinia 管理全局认证状态，Axios 拦截器统一处理 Token 注入与 401 自动跳转
- 集成高德地图 API 实现天气自动获取，作为 Agent 工具和日记上下文的一部分丰富 AI 分析维度
- 实现标签管理系统（自定义标签、颜色、引用计数），标签作为 Few-shot 上下文注入 Prompt，帮助 AI 理解日记主题与情感

项目亮点：
- ReAct Agent + RAG 语义检索的 AI 架构设计，实现了从"字面匹配"到"语义理解"的检索升级
- 多租户数据隔离贯穿全链路（MySQL 行级隔离 + Chroma Collection 级隔离 + JWT 身份校验）
- 支持用户自定义 LLM 模型配置，兼容所有 OpenAI API 格式的服务商
- Token 消耗精细化追踪，区分缓存命中（免费）与实际付费部分，帮助用户控制成本
