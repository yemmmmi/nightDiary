夜记 — AI 智能日记管理系统

技术栈： Vue 3 + TypeScript + Tailwind CSS + Pinia | Python + FastAPI + SQLAlchemy + MySQL | LangChain + Chroma 向量数据库 | Redis（缓存/分布式锁/限流） | JWT + bcrypt

项目描述： 一款支持多用户的全栈 Web 日记平台，核心为基于 ReAct Agent 架构的 AI 日记分析系统。用户可记录日记、管理标签、配置个人 LLM 模型，
系统通过 LangChain ReAct Agent 结合 RAG 语义检索对日记内容进行深度分析并生成回应。支持公开日记专栏、天气自动获取与缓存预加载等功能。

核心职责与实现：

设计并实现 ReAct Agent 架构的 AI 分析引擎，集成 LangChain 工具链（日记语义搜索、天气查询、情感分析、地址获取），根据日记长度和历史数量智能切换 Chain/Agent 模式，优化 Token 消耗
基于 Chroma + text2vec-base-chinese Embedding 构建 RAG 检索系统，按用户隔离 Collection，实现语义级历史日记检索，替代传统 SQL LIKE 关键词匹配
实现多 LLM 提供商适配（DeepSeek、OpenAI、通义千问等），用户可自行配置模型，API Key 采用 Fernet 对称加密存储，任何接口不返回原始密钥
基于 Redis 实现多层缓存架构：Cache-Aside 旁路缓存 + 空值缓存防穿透 + 分布式锁防击穿 + 随机 TTL 防雪崩 + 滑动窗口 IP 限流；Redis 不可用时全链路降级
实现天气缓存预加载机制，用户登录后自动预热天气缓存，地址变更时自动失效旧缓存并异步预加载新数据，消除日记发布时的等待
设计公开日记专栏系统，支持日记发布/下架、分页列表、详情查看，配合 Redis 缓存策略保证高并发下的读取性能
设计智能防重分析机制，通过日记内容长度比对避免重复调用 LLM；记录详细 Token 消耗明细（缓存命中/付费输入/付费输出）
前端采用 Vue 3 Composition API + TypeScript，Pinia 管理全局状态，Axios 拦截器统一处理 Token 注入与 401 跳转；实现天气组件根据天气类型自动切换主题风格
项目亮点：

ReAct Agent + RAG 语义检索的 AI 架构
Redis 多层缓存防护体系（穿透/击穿/雪崩）+ 分布式锁 + 滑动窗口限流，全链路降级兜底
多租户数据隔离贯穿全链路（MySQL 行级隔离 + Chroma Collection 级隔离 + JWT 身份校验）
支持用户自定义 LLM 配置，兼容所有 OpenAI API 格式服务商；
Token 消耗精细化追踪，区分免费缓存与实际付费部分
